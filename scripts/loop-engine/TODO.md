# TODO — ループが消化していく作業リスト

ループ（や run-once.sh）は、この中の **未完（- [ ]）を上から1つだけ**選んで実装する。
完了したら AI 自身が `- [x]` に変える。状態をこのファイルに「外出し」しておくことで、
AI の記憶（コンテキスト）が毎回リセットされても作業を続けられる。

## 使い方

- 1行 = 1つの小さく完結するタスク。大きすぎる項目は分割する。
- 空欄のときは run-once.sh / loop.sh は「やることなし」で安全に終わる。
