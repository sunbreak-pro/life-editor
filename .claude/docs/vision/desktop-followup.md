# Desktop Followup

> Desktop は主目的（Daily / Note / Schedule table の読み書き）をほぼ完遂。以下は **普段使いながら追加** する残課題。

---

## 残課題

- **Materials**: File タブの動作詳細化（操作粒度・ショートカット・プレビュー挙動）
- **Notes (Node)**: 編集体験・接続 UI・階層操作の磨き込み
- **Board (Paper Boards)**: Tier 3 凍結の見直し余地（Note connections との棲み分けを踏まえて判断）

## 方針

- 新規大型機能は立てない。既存 3 本柱（Daily / Note / Schedule）を壊さないこと最優先
- MCP ツール拡張は主目的に不要。必要が発生したときのみ検討
- 個別実装は `.claude/YYYY-MM-DD-<slug>.md` プランで扱う（本ファイルは方針メモ）
